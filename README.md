# Accessibility Persona Card Website

This project is an inclusive persona card website designed for a **high school student with autism and color blindness**, with a strong focus on accessibility, readability, and reduced cognitive load.

The website applies established accessibility guidelines and reflects critically on existing persona design frameworks to propose improvements for users with autism and/or color blindness.
